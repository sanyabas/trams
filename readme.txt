Приложение "Где трамвай"

Поддерживаемые функции:
-просмотр карты
-отображение всех трамваев на карте
-перемещение карты

Требования:
Python версии 3.5
PyQt5

Использование:
пример запуска: python trams.py

Описание интерфейса:
Кнопка "Close" закрывает приложение
Кнопка "Refresh" обновляет местоположение трамваев
Кнопки с номерами маршрутов позволяют выбирать отображаемые маршруты (не реализовано)
